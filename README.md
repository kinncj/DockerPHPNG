### Docker PHPNG

#### Usage

###### Default usage
```
git clone git@github.com:kinncj/DockerPHPNG.git

sudo docker run -d -P DockerPHPNG installer
```

###### Specify a configuration file
```
git clone git@github.com:kinncj/DockerPHPNG.git

sudo docker run -d -P DockerPHPNG installer http://path.com/to/config.json
```
